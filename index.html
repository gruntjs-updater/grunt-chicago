Installation

Using npm

npm install grunt-chicago --save-dev

Usage

Basic usage.

In your project's Gruntfile, add a section named chicago to the data object passed into grunt.initConfig():

grunt.initConfig({
	chicago: {
		your_target: {
			// Target-specific file lists and/or options go here.
		},
	},
});

To inject or import files in your project, add a line that contains the following to any JavaScript file:

// @import path/to/import.js

Set just the source files to overwrite the existing import statement:

grunt.initConfig({
	chicago: {
		files: {
			src : [
				'src/file1.js',
				'src/file2.js'
			]
		},
	},
});

Or using globbing patterns to write the output to a new file:

grunt.initConfig({
	chicago: {
		files: [{
			expand: true,
			cwd: 'src',
			src: ['*.js', '!*.min.js'],
			dest: 'dest',
			ext: '.js',
			extDot : 'last'
		}],
	},
});

And execute the task in Terminal:

grunt chicago

Options

All plugin & event options.
